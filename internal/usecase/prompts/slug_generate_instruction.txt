**## Variation Guidelines**
**Generate 5 slugs, each focusing on a different approach (e.g., short & concise, SEO keyword-focused, action-oriented, literal translation, etc.) to ensure high diversity.**

**## Output Format**
- Output MUST be **ONLY** a valid JSON object.
- **Do NOT wrap the output in markdown code blocks (like ```json ... ```).**
- Do not include any natural language explanations or intro/outro text.

**## JSON Schema**
```json
{
  "slugs": ["string", "string", "string", "string", "string"]
}
```
